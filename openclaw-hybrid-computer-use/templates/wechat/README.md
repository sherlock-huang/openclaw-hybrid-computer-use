# WeChat UI 模板库

此目录存放微信界面元素的截图模板，供 `AdaptiveCoordinateMapper` 在比例坐标失效时进行模板匹配 fallback。

## 所需模板文件

| 文件名 | 对应元素 | 截图建议 |
|:---|:---|:---|
| `wechat_search_box.png` | 顶部搜索框 | 截取搜索图标 + "搜索" 文字区域 |
| `wechat_input_box.png` | 聊天输入框 | 截取底部 "发送消息" 提示文字区域 |
| `wechat_send_button.png` | 发送按钮 | 截取输入框右侧的蓝色/绿色发送按钮 |

## 制作模板的方法

1. 打开微信窗口，调整到常用尺寸（如 1000x700）
2. 使用 Windows 截图工具（Win+Shift+S）精确截取目标元素
3. 保存为 **PNG** 格式到此目录
4. 建议尺寸：宽度 80~200px，高度 20~60px
5. 建议包含少量周围背景（5~10px），但不要包含动态内容（如头像、消息内容）

## 注意事项

- 模板匹配对颜色和尺寸敏感，建议在不同主题下分别制作模板
- 如果微信更新后 UI 发生变化，可能需要重新截取模板
- 模板不存在时，系统会自动回退到比例坐标法
